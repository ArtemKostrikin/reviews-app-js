# Приложение отзывов (JS)

Простое приложение на JavaScript для добавления и управления отзывами о продуктах.

## Возможности
- Добавление отзывов
- Просмотр отзывов по продуктам
- Удаление отзывов
- Сохранение данных в localStorage

## Технологии
- HTML
- CSS
- JavaScript (vanilla)
- localStorage

## Запуск
Открыть файл `index.html` в браузере.
